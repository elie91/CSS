Basic Customization

How To Edit Files

You can use whatever text editor you like, even default Text Editor in the OS. But we recommend to use the one with th syntax highlighting such as:

  • Sublime
  • Notepad++

How To Change Fonts

  1. If you are not satisfied with the default font, you can choose from hundreds ones from the Google Fonts. After you are happy with the selected font, click SEE SPECIMEN. You'll be redirected to to font detail. Then click Select this font. Popup window will show with text "1 Family Selected" - click it. Then you can customize your selection by clicking Customize. When you are done copy the text in import tab.
    For example @import url('https://fonts.googleapis.com/css?family=Open+Sans|Poppins:500,700');
  2. Then open the main.css file to change fonts and find the @import... element which loads the Google font and place your copied code.
  3. Final step is to change the font family in the css/main.css file. Find body tag on the beginning of the file and font-family inside. Go back to Google Fonts page where you have chose your font and copy the line placed in the Specify in CSS. In this example it is font-family: 'Open Sans', sans-serif; and also do the same for h1,h2,h3 ... tag.

Advanced Customization

JavaScript

JavaScript (JS) files/plugins are located in the /js folder and in the HTML file they are called using code. All JS files are located at the bottom of the HTML file. main.js file is the template's JS file that includes scripts for running other plugins.

Note: You will need to modify only main.js file.

Forms

Template uses Bootsrap form structure and brings some own styles.

Contact Form

Space comes with working contact form. You can use them for contact or other forms that will collect the user's email address and send it to your receiving one.

Note: It is better to test email sending on live server. It does not work on a local machine also make sure phpmailer funtion is properly working on server.

Setup a Receiving Email Address

You have to setup your own email address for email receiving. As a default it is email@gmail.com and it is located in the assets/php/mail.php.
How to change the address:

Go to assets/php/mail.php and find $recipient = "email@gmail.com"; and change the address to your own.
